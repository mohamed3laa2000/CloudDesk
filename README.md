# CloudDesk EDU

A professional enterprise UI demo for a cloud desktop platform targeted at students, lecturers, and professionals.

## UI Vision

- **Enterprise confidence**: Stable, trustworthy, professional interface
- **Clarity over cleverness**: Every element serves a purpose
- **Spatial breathing room**: Generous padding and whitespace
- **Purposeful hierarchy**: Clear visual weight guides the eye
- **Accessible by default**: High contrast, readable type, keyboard-friendly

## Design System

**Colors:**
- Primary: Indigo (#4F46E5)
- Neutrals: Gray scale from 50 to 900
- Background: White (#FFFFFF) and Gray-50 (#F9FAFB)

**Typography:**
- Font: Inter
- Body: 14px, line-height 1.5
- Headings: 24px (page), 18px (section), 16px (subsection)

**Layout:**
- Left sidebar (240px): Primary navigation
- Top bar (64px): Breadcrumbs and global actions
- Main content: Max-width 1280px, generous padding

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Lucide React (icons)
